# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in iyzico-js, please report it through [GitHub Private Vulnerability Reporting](https://github.com/feroxbyte/iyzico-js/security/advisories/new).

**Please do NOT open a public issue for security vulnerabilities.**

## Response

We will acknowledge your report within **48 hours** and work with you to understand and address the issue.

## Scope

This policy covers the iyzico-js client library itself. For vulnerabilities in the iyzico API or payment infrastructure, please contact [iyzico](https://www.iyzico.com/) directly.
